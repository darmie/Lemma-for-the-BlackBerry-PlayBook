Actions

Touch the actions icon () beside a tweet if you'd like to:
  • View In App the link contained in a tweet (only visible when a tweet contains a URL)
  • Reply to a tweet by writing a public @username tweet
  • Retweet a tweet
  • Retweet With Message, which gives you the option to add or modify text before tweeting
  • Send Direct Message to the user of the tweet (only works if that user is currently following you)
  • Favourite the selected tweet
