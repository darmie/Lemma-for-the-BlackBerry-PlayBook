About Lemma
Lemma for the BlackBerry PlayBook is a free, open-source application that was written as a proof of concept of the BlackBerry PlayBook WebWorks SDK (HTML5 and JavaScript). The code for Lemma for the BlackBerry PlayBook can be downloaded from GitHub to be updated, optimized and enhanced by the thriving BlackBerry WebWorks developer community.

To read more about Lemma, visit http://lemmaforplaybook.com/ .

To reach the primary contributor to Lemma, contact Sully Syed (@sullysyed) on Twitter.
